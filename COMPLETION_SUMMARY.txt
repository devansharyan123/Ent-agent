🎯 HOW IT WORKS (3 Steps):

1. User asks question via /ask endpoint
2. Query passed to policy_retrieval_tool which:
   - Enforces RBAC (SQL WHERE clause)
   - Generates embedding
   - Performs vector search
   - Calls Groq LLM (temperature=0.0)
3. Returns grounded answer + sources

Example Flow:
  Employee asks "What is maternity leave?"
  → RBAC check: allowed_categories = ["general"]
  → SQL: WHERE category = ANY(['general'])
  → Returns ONLY general chunks
  → HR/Admin policies NEVER returned
